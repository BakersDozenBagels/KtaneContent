Pride in Purple

Solution

The flavour text and title hint at the modules Red and Blue, therefore we have to use their manuals to solve this puzzle.

Each line uses the following coventions:

  • Warm/Cold — Warm refers to Red's manual, Cold refers to Blue's manual.
  • Red/Yellow/Green/Blue — Refers to a colour's corresponding cell in the previously referred manual (eg. Warm Yellow refers to Yellow's cell in Red's manual).
  • - N — Subtract N from the obtained value, where N is the given number.

As hinted at by the flavour text, take the largest time from each colour's corresponding cell and subtract N [ignore "0:" before any times]. By converting each of these values into letters using A1Z26, we get the answer to the puzzle.
