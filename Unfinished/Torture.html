On the Subject of Torture

*sighs* “Here we go again.”

This module consists of a 4×4 grid with a chessboard-like pattern, of black and azure tiles.

Pressing any tile will add some offset to 3 or 4 other tiles, which can potentially be different per tile affected. For example, pressing A1 can add 4 to C2, 7 to D4, and 3 to D1.

To disarm this module, press the tiles some number of times, such that, in the end, each tile displays the same number.
