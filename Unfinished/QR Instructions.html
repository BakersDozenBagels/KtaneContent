Section M4TH: Math Information

The QR code displays 3 single-digit numbers. Use the following rules to change the numbers, then solve the module. Submitting an incorrect answer will cause a strike.

The first, second and third numbers will be referred to as "X", "Y" and "Z" respectively.

First Number:

  • Multiply this number by the amount of port plates.
  • Add the amount of ports to this number.
  • Subtract 10 from this number if the serial number has a vowel.
  • Add 25 for every unlit indicator.

Second Number:

  • Add all of the digits in the serial number to this number.
  • If x is an even number, add 137 to this number.
  • Multiply this number by the amount of letters in the serial number.
  • Subtract 50 if the bomb has a lit BOB indicator.
  • If X + Y = prime number, double this number.

Third Number:

  • Multiply the number by the amount of battery holders.
  • If Y - X = prime number, triple this number.
  • Add 42 to this number.

Add X, Y and Z together, then multiply this number by the amount of modules on the bomb. This is your answer.

If the number is at least four digits long, submit the 3 least significant digits.

To submit the answer, press the correct buttons when the last digit of the bombs timer is a number you need, then press the submit button.

Button A changes the units, button B changes the tens, button C changes the hundreds and button D submits the answer.

Example: If the answer is 523, you press the button that changes the hundreds number when the last digit of the timer is 5. Do the same with the other numbers until you have your answer.

Section D1G1T4LR00T: Digital Root Information

The QR code displays a ten-digit number. Use the following rules to change the number, then submit it. Pressing an incorrect button will cause a strike.

  • If there are 3 or more letters in the serial number, triple the number.
  • Subtract 9001 for every lit indicator.
  • Subtract 11111 for every distinct port type.
  • Add 800008 for every battery holder.
  • If the number is now even, halve it.

Take the digital root of this number, then convert all the serial number letters to numbers (A=1, B=2, etc), then add them to the number. Take the digital root of this number. This is your answer.

Section B1N4RY: Binary Information

The QR code displays a number in binary. Use the following rules to change the number, then submit it. Submitting an incorrect answer or pressing the strike button will cause a strike.

  • Add all of the digits of the binary form of the number, letters in the serial number, distinct port types and port plates to the number.
  • If the number is now odd, double it.
  • Subtract 2 for every lit indicator present.
  • Add 3 to the number if the last digit of the serial number is even. Add 8 if it is even.
  • Subtract 13 for every vowel in the serial number.
  • If the number is now even, double it.
  • If there is at least 1 battery holder, multiply the number by the amount of batteries.

If the answer is nevative or greater than 15, repeatedly add or subtract 5 until the answer is between 0 and 15.

Submit the answer in binary.

Section G0N3: No Information

The QR code displays nothing. Use the following rules to make your own information and decide the correct section to use.


If the bomb has no batteries: You are solving a maze puzzle. The first letter and number in the serial number is your first coordinate, and the second letter and number is your second.

If the bomb has more indicators than batteries: You are solving a math puzzle. The number of distinct port types is your first number, the amount of letters in the serial number is your second, and the last digit of the serial number is your third.

If the bomb has More numbers than letters in the serial number: You are solving a digital root puzzle. Your number is 2018133727.

If the bomb has no ports: You are solving a binary puzzle. Your number is the amount of batteries multiplied by the amount of battery holders. If this number is greater than 15, modulo 5 until it is less than 16.

Otherwise: You are solving a Morse puzzle. Your word is "--.- ..- .. -".
