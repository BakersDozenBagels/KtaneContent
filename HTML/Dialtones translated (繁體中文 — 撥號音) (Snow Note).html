Keep Talking and Nobody Explodes Mod 撥號音

關於【撥號音】模組

求求你，求求你，求求你請聽我說。

模組上有0~9、播放鍵和錄音鍵。按下播放鍵可以播放一串撥號音訊。只要不按下錄音鍵，可以自由按下各數字鍵和播放鍵。

撥號音訊序列以下列方式進行變換。

  • 撥號鍵所按下的次數 n ，轉換為各撥號鍵的下方從左邊數起第 n 個英文字。(例:按 3 次【5】等於 L )
  • 【1】用作分割同一按鍵的英文字 (例:551555為KL)。

要拆除模組，在表中找出播放鍵所表示的單詞，然後以文字讀順方式移動【序號末位數字】次。右下的單詞的下一個為左上角的單詞。

找出要提交的單詞，按下錄音鍵以開始輸入。單詞必需以以上的形式提交(也就是說一串數字)。輸入後再按一次錄音鍵以提交。如果提交錯誤的單詞，會觸發一次失誤並更換一個新的單詞。按下錄音鍵後，除了提交以外不能刪除先前的輸入。
