モジュール詳細：そんなにビッグじゃないサークル

フラフープをモジュールにするというアイデアを思い付いたのは誰だ？

  • BOB/CAR/CLR : 点灯…+1/消灯…-1
  • FRK/FRQ/MSA/NSA : 点灯…+2/消灯…-2
  • SIG/SND/TRN : 点灯…+3/消灯…-3
  • 解除モジュール1つにつき3を加算
  • バッテリー : 奇数…+4/偶数…-4
  • パラレル : 単体…+5/シリアルあり…-4
  • DVI-D : 単体…-5/RCAあり…+$
  • 各改造インジケーター = +6
  • 各改造ポート = -6
  • 2ファクタの最下位桁を追加
  • 絶対値を取得
  • 最下位桁を取得
  • 表では数字0を-0-と表記
    • その位置にあるシリアルナンバーを取得
      • 0 = 1番目
      • 1/9 = 2番目
      • 2/8 = 3番目
      • 3/7 = 4番目
      • 4/6 = 5番目
      • 5 = 6番目
